## 🚀 Деплой на Render.com (Бесплатно)

### Шаг 1: Подготовка репозитория

```bash
# 1. Создай папку проекта
mkdir maxis-strike
cd maxis-strike

# 2. Создай файлы (или скопируй из этого репозитория)
# - server.js
# - index.html  
# - package.json
# - README.md

# 3. Инициализируй git
git init
git add .
git commit -m "Initial commit: MAXIS STRIKE v3.0"

# 4. Создай репозиторий на GitHub
# Перейди на github.com/new
# Название: maxis-strike
# Создай репозиторий

# 5. Загрузи на GitHub
git remote add origin https://github.com/YOUR_USERNAME/maxis-strike.git
git branch -M main
git push -u origin main
```

### Шаг 2: Деплой на Render.com

1. **Зайди на [render.com](https://render.com)** → Sign Up (бесплатно)

2. **Создай новый Web Service:**
   - Нажми **New +** → **Web Service**
   - Подключи GitHub → выбери репозиторий `maxis-strike`

3. **Настройки сервиса:**

   | Параметр | Значение |
   |----------|----------|
   | **Name** | `maxis-strike` |
   | **Build Command** | `npm install` |
   | **Start Command** | `npm start` |
   | **Instance Type** | `Free` (бесплатный) |
   | **Region** | `Oregon` (или любой) |

4. **Нажми Create Web Service**

5. **Жди 2-3 минуты** пока Render соберёт проект

### Шаг 3: Играй!

После деплоя получишь URL вида:
```
https://maxis-strike.onrender.com
```

Открой в браузере → **PLAY** → Создай сервер или подключись!

## 🎮 Как играть

### Создание сервера:
1. Нажми **PLAY** в главном меню
2. Введи название сервера
3. Пароль (опционально)
4. Выбери карту: **MS_DUST** или **MS_START**
5. Выбери режим: **Team Deathmatch** или **Hostage Rescue**
6. Нажми **CREATE SERVER**

### Подключение:
1. Выбери сервер из списка
2. Нажми **JOIN SERVER**
3. Если есть пароль — введи его

### Управление:

| Клавиша | Действие |
|---------|----------|
| `W A S D` | Движение |
| `Мышь` | Обзор |
| `ЛКМ` | Стрельба |
| `R` | Перезарядка |
| `Space` | Прыжок |
| `1` | AK-47 (авто) |
| `2` | Desert Eagle |
| `3` | Knife (без патронов!) |
| `E` | Спасти заложника (CT) |
| `Tab` | Скорборд |
| `T` / `Y` | Чат |
| `Esc` | Меню / Выйти |

### Мобильное управление:
- **Левая сторона** — кнопки движения (◄ ▲ ▼ ►)
- **Правая сторона** — кнопки действий (🔫 ↑ R 1 2 3)
- **Верх экрана** — прицеливание (перетаскивание)

## 🗺 Карты

### MS_START (Арена)
- Маленькая симметричная карта
- Идеально для быстрых боёв
- T слева, CT справа
- Центральная структура

### MS_DUST (Пустыня)
- Большая карта в стиле de_dust
- Два сайта: A и B
- Больше укрытий и тактики
- Песчаная текстура

## 🎯 Режимы игры

### 💀 Team Deathmatch
- Террористы vs Спецназ
- Убийство = +1 очко команде
- Респавн через 3 секунды
- Побеждает команда с больше очков

### 🧑‍🤝‍🧑 Hostage Rescue
- CT должны спасти 2 заложников
- Подойди к заложнику + нажми **E**
- Доведи заложника до зоны CT (спавн)
- Если все спасены — CT побеждают!

## 🎯 Технические детали

### Сервер (server.js):
- **HTTP Server** — отдаёт index.html
- **WebSocket Server** — мультиплеер синхронизация
- **Состояние игры** — комнаты, игроки, заложники
- **Античит** — проверка скорости, телепорта, частоты
- **Лидерборд** — глобальная статистика
- **Достижения** — серверная проверка

### Клиент (index.html):
- **Three.js** — 3D рендеринг
- **Web Audio API** — звуки выстрелов, шагов
- **Canvas** — текстуры генерируются на лету
- **CSS** — UI, эффекты, анимации
- **WebSocket** — связь с сервером

### Производительность:
- **50 FPS** — синхронизация состояния
- **16 игроков** — максимум в комнате
- **Free план** — 512MB RAM, 1 CPU

## 🔧 Отладка

### Сервер не запускается:
```bash
# Проверь порт
lsof -i :3000

# Убей процесс
kill -9 <PID>
```

### WebSocket не подключается:
- Проверь, что сервер запущен
- Проверь консоль браузера (F12)
- Проверь CORS (не нужен при одном домене)

### Игроки не видят друг друга:
- Убедитесь, что все на одном сервере
- Проверьте WebSocket соединение
- Обновите список серверов (Refresh)
